## Counter Application

CounterApp은 매우 간단한 숫자 세기 어플입니다.
일상생활을 하다가 숫자를 세야 할 필요가 있을 경우 활용이 가능합니다.
숫자를 세다가 자꾸 깜빡 하시는 분들 이용해보시기 바랍니다.

## 기능
* 숫자 업다운
* 버튼 홀드
* 리셋
